# node-vue-vite
node + vue3 + vite 

# Vue 项目启动 目录 upload-vue
```javascript
  yarn dev
```

# Vue 项目打包 目录 upload-vue
```javascript
  yarn build
`````

# node 服务启动 目录 node-express
```javascript
  node server.js
```
